# BTS 소개 웹 페이지 제작 계획

## 기술 선택: 순수 HTML/CSS/JS (단일 파일)

별도 빌드 도구나 패키지 설치 없이 `index.html` 하나로 완성합니다.

- 빌드 환경 불필요, 브라우저에서 바로 열기 가능
- BTS 공식 컬러(보라색 계열 `#7B2D8B`, `#4A0E8F`) 기반 디자인
- CSS 애니메이션 및 스크롤 인터랙션 포함
- 완전 반응형 (모바일/태블릿/데스크탑)

## 페이지 구조

```
index.html
```

### 섹션 구성

1. **Hero** — 풀스크린 배경, BTS 로고, 소개 문구, 부드러운 페이드인 애니메이션
2. **About BTS** — 결성 연도(2013), HYBE(빅히트), ARMY 팬덤 등 간략 역사
3. **Members** — 7명 카드 그리드 (RM, Jin, SUGA, j-hope, Jimin, V, Jung Kook) — 이름/포지션/생년월일
4. **Discography** — 주요 앨범 타임라인 (Dark & Wild → Love Yourself 시리즈 → Map of the Soul → Butter → Proof 등)
5. **Achievements** — 빌보드 Hot 100 1위, UN 연설, Grammy 노미네이션 등 주요 수상/업적 카드
6. **Gallery** — CSS 그리드 기반 이미지 플레이스홀더 (Unsplash 무료 이미지 URL 사용)
7. **Social Links** — YouTube, Instagram, Twitter/X, TikTok, Weverse 링크 버튼
8. **Footer** — 저작권 표시

## 디자인 포인트

- 배경: 딥 퍼플-블랙 그라디언트
- 타이포그래피: Google Fonts (Noto Sans KR + Playfair Display)
- 호버 효과: 카드 transform/shadow 트랜지션
- 스크롤 애니메이션: Intersection Observer API로 섹션 페이드인
